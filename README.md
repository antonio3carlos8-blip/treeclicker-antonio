# 🐺🌕 Projeto Interativo com Next.js

Este é um projeto desenvolvido com Next.js que apresenta um cenário visual interativo com elementos personalizados.

## 🚀 Funcionalidades

- 🐺 Inserção de um lobo no cenário
- 🌕 Adição de uma lua para ambientação
- 🎨 Melhorias na interface e organização visual
- 🔄 Alteração do tema original (remoção da árvore)

---

## 🛠️ Tecnologias utilizadas

- Next.js
- React
- Tailwind CSS

---

## ▶️ Como rodar o projeto

Execute o servidor de desenvolvimento:

```bash
npm run dev
